# Погодный Аналитик

Анализ исторических температурных данных и мониторинг текущей погоды через OpenWeatherMap API.

## Функции
- Анализ временных рядов (скользящее среднее, аномалии, тренды)
- Параллельная обработка данных
- Сравнение синхронных и асинхронных API-запросов
- Интерактивные графики (Plotly)
- Проверка текущей температуры на аномальность
